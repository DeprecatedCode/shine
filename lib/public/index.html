Shine Network

Use the dev console to experience Shine.

shineClient.help;

 

Setup:

var socket = io.connect();

var shineClient = shine.client(socket);

var registry = shineClient.app('@registry');

var anon = registry.resource('user.@anonymous');

var store = shineClient.app('@public').as(anon);

 

Operation:

store.resource('key').put({"complex": "value"});

store.resource('key').get(function (value) { console.log(value); });
